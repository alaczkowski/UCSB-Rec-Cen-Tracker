# Rec Cen Tracker

A simple React/Vite website for sharing UCSB Rec Cen crowd insights.

## How to run locally

1. Install Node.js.
2. Open this folder in VS Code or Terminal.
3. Run:

```bash
npm install
npm run dev
```

## How to deploy on Vercel

1. Upload this whole folder to a GitHub repository.
2. Sign in to Vercel with GitHub.
3. Import the GitHub repository.
4. Click Deploy.
5. After Vercel gives you a real URL, replace the `siteUrl` value inside `src/App.jsx` with that real link so the QR code points to the correct website.

## Important note

The current check-in/check-out buttons are a front-end demo only. They do not save shared live data yet.
